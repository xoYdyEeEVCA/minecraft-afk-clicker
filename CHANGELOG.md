# Changelog

## [v2.4.1] - 2023-10-10
### Added
- Implemented **Potion Support**: Users can now select different potions to enhance AFK activities, including Speed and Strength potions.
- New **Auto-Clicking Modes**: Introduced a **Sneak Click** mode for stealth gameplay, especially useful for PvP scenarios.
- Added support for **Minecraft Bedrock Edition** in addition to Java Edition.

### Changed
- Enhanced GUI for better user experience, including larger buttons and clearer information display on clicking statistics.
- Optimized the **Farm Auto-Clicker** feature to improve efficiency when collecting crops, like Wheat and Carrots.

### Fixed
- Resolved a bug where the app would crash when switching between profiles with different hotkey configurations.
- Addressed an issue where the click timer would not reset properly after completing automated tasks.

---

## [v2.3.0] - 2023-08-15
### Added
- Introduced **Multi-Action Profiles** that allow users to set specific tasks for different Minecraft activities (e.g., mining, building).
- Users can now **map hotkeys** to specific actions, allowing for tailored gameplay experiences.

### Changed
- Updated **Auto-Mine feature** to include support for more block types, enabling users to automatically mine rare ores like Diamonds and Emeralds.
- Improved the performance of the click simulation engine for a more realistic and responsive experience.

### Fixed
- Fixed an issue where the app would fail to recognize multiplayer server connections leading to click disruptions.
- Resolved a bug that prevented users from selecting a skin for their AFK avatar.

---

## [v2.1.0] - 2023-06-01
### Added
- Integrated **Enchanting Station** feature to allow automated enchanting of tools and armor while AFK.
- Added a **Custom Click Sound**, allowing users to choose sounds from the game for a more immersive experience.

### Changed
- Redesigned the settings menu to provide clearer options for configuring click speeds and actions.
- Updated all icons to reflect a more modern look and feel that matches Minecraft’s aesthetic.

### Fixed
- Fixed various minor bugs that caused unexpected behavior during extended AFK sessions.
- Addressed a recurrent issue with system resource usage when multiple instances of the app were running.
